Tags

As tags em HTML são muito importantes para ajudar na personalização e estilização dos textos e aplicar funcionalidades.

Tags H1 ao H6

São tags utilizadas para titulos e definem o seu tamanho, sendo H1 a maior e H6 a menor.

Tag mark

Tag utilizada para destacar um pedaço de texto. Exemplo de texto marcado.

Tag p

Tag para definir o paragrafo do texto. Possui espaçamentos predefinidos ao redor do texto com a tag.

Tag small

Essa tag diminui o tamanho do texto.

Tag i

Tag para deixar o texto em italico.

Tag u

Tag para sublinhar o texto.

Tag strong

Tag para deixar o texto em negrito.

Tag ol

Essa tag cria uma lista ordenada, numerando cada item da lista.

Tag ul

Essa tag cria uma lista não ordenada, cada item da lista é incluído sem numeração ou outra forma de ordenação.

Tag li

Tag para definir como item da lista.

Tag a

Tag para linkar o texto para páginas externas da internet ou ancorar para sessões da página atual.

Tag hr

Essa tag cria um linha para marcar uma divisória na página.

Tag sub

O texto ficará um pouco abaixo da altura da linha.

Tag sup

O texto ficará um pouco acima da altura da linha.

Tag blockquote

Deixará o texto como uma citação.
